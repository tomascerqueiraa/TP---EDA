Deduz os efeitos nefastos com base nas posições das antenas.

Esta função percorre a lista de antenas e verifica todos os pares de antenas para calcular os efeitos nefastos entre elas. Para cada par de antenas, se as condições forem atendidas (mesma frequência e distância adequada), um novo efeito é gerado e adicionado à lista de efeitos.
